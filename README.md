🚆 Railway Reservation System (C++)
📌 Project Overview

onlineGDB = https://onlinegdb.com/ZvIjWVgPp

This project is a simple Railway Reservation System developed in C++ using Object-Oriented Programming (OOP) concepts.
It allows users to manage train records such as adding, displaying, and searching trains.

🎯 Features

➕ Add new train details
📋 Display all train records
🔍 Search train by train number
📊 Count total number of trains using static member
🧱 Concepts Used

Classes & Objects
Constructors (Default, Parameterized, Copy)
Destructor
Static Data Members
Vectors (STL)
Encapsulation
🏗️ Class Structure
🚉 Train Class

Stores details of a train:

Train Number
Train Name
Source
Destination
Train Time

Functions:

inputTrain() → Take input from user
displayTrain() → Show train details
getTrainNumber() → Return train number
getTrainCount() → Static function to count trains
🚦 RailwaySystem Class

Manages multiple trains using a vector.

Functions:

addTrain() → Add a new train
displayAll() → Show all trains
searchTrain(int num) → Search train by number
